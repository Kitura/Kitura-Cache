KituraCache

public class KituraCache

A thread-safe, in-memory cache for storing an object with a Hashable key. Time To Live can be specified per entry, or will take a default value specified when the cache is initialised.

    Declaration

    Swift

    public init(defaultTTL: UInt = 0, checkFrequency: UInt = 600)

    Parameters

    defaultTTL

    The default Time to Live value (in seconds). Used for cache entries for which the TTL is not specified otherwise. If it is not specified here, the default value is 0 which maps to infinity.
    checkFrequency

    The frequency (in seconds) to check for expired entries. If it is not specified here, the default value is 600 (10 minutes).

    Usage Example:

    //In this case, item has been stored in the cache with an Int key

...

    //Object with key of 1 retrieved from cache.
    ...
}
else {
    //No object stored in cache with key of 1.
    ...
}

    Note

    The return value will be nil if there is no object in the cache with the specified key.
